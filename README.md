# dotsfile

## create_symlink

```bash
chmod +x ./scripts/create_symlink.sh
```

上記で、シンボリックリンクを作成できる

## win32yank

https://github.com/equalsraf/win32yank

-   win32yank-x64.zip をダウンロード
-   win32yank.exe の取り出す

```bash
sudo mv win32yank.exe /usr/local/bin
sudo chmod +x /usr/local/bin/win32yank.exe
```
